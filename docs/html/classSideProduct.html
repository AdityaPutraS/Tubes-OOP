Detailed Description

SideProduct adalah base class dari segala SideProduct (produk olahan hasil mix) di game.

SideProduct dihasilkan saat player berinteraksi dengan Mixer dan mencampur 2 atau lebih FarmProduct Harga SideProduct ditentukan secara manual
