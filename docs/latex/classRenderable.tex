\doxysubsection{Detailed Description}
Class sebagai base semua class yang bisa di render. 

Class ini dibuat dengan tujuan mengabstraksi sistem render semua objek dengan memanggil class tampilan dan mengeprint char\+Representasi di posisi x\+Layar, y\+Layar
